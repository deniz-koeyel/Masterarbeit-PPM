# Predictive Process Monitoring: TabPFN vs. XGBoost

Dieses Repository enthält den vollständigen digitalen Anhang (Quellcode und Jupyter-Notebooks) zur Masterarbeit:
**"Predictive Process Monitoring mit Foundation Models und klassischem Machine Learning: Eine empirische Vergleichsstudie"**

Autor: Deniz Köyel  
Institution: FOM Hochschule für Oekonomie und Management  
Jahr: 2026

## 📌 Kontext der Arbeit
Die Arbeit untersucht die algorithmischen Limitierungen von Kaltstart-Szenarien (Small Data) im Predictive Process Monitoring. Dabei wird das klassische Gradient Boosting (XGBoost) gegen das Transformer-basierte Foundation Model (TabPFN) evaluiert. Die Experimente umfassen sowohl unstrukturierte (Low-Effort) als auch domänenspezifische (High-Effort) Feature-Räume.
